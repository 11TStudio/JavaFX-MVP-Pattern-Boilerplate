## About
An simple JavaFX project example using the latest MVP patterns. As extra added a Log class so you can use.


### Prerequisites
 * Git
 * JavaFX SDK 11.02 (tested with this)
### Installation
1. Clone the repository.
2. Open the project from your IDE
3. Go to `File>Project Structure>Global Libraries` and add your global javafx library.
3. Run Main (if it doesnt work run MainHack)
